# Learn React By Example - Generate UUID App

This repo has two apps:
- Vanilla app
- React app

## To run vanilla app:

```
cd vanilla-app
npm i
node index.mjs
```

Open a browser with URL http://localhost:8080.

## To run React app:

```
cd react-app
npm i
npm start dev
```

Open a browser with URL http://localhost:3000
